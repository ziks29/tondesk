## 2024-04-29 - Accessible Interactive Components
**Learning:** Custom interactive elements like switches and accordions built with standard HTML tags require manual ARIA state bindings (role="switch", aria-checked, aria-expanded, aria-controls) to be properly interpreted by screen readers.
**Action:** When creating custom interactive UI patterns, always cross-reference WAI-ARIA guidelines to ensure required roles and state attributes are applied dynamically based on component state.
